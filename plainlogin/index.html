Single page app login example

This page tests "vanilla" login (and nothing else) from a single page app. signIn() and signOut() functions are defined in authRedirect.js file. Notice: I've hardcoded signIn() callback function in authRedirect.js as loginCallback(), which is implemented in this index.html file. You may want to change that in your own implementation.
